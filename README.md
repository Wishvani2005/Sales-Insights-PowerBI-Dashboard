# 📊 Sales Insights Dashboard (Power BI)

## 🚀 Project Overview
This is an end-to-end Personal Business Intelligence project focused on analyzing sales performance, tracking key growth metrics, and delivering actionable insights for data-driven decision-making. The entire project workflow—from data cleaning to dashboard implementation—was executed independently.

## ✨ Key Features & Functionalities
* **Dynamic KPI Tracking:** Implemented custom **DAX measures** to calculate and display monthly revenue growth, profit margins, and overall sales volume.
* **Interactive Visualizations:** Integrated clean monthly trend charts and interactive regional revenue maps to pinpoint high-performing markets.
* **Data Modeling & ETL:** Cleaned, transformed, and structured raw source datasets using **Power Query** to ensure a high-performance star schema model.
* **User-Centric UI/UX:** Designed a clean, professional dashboard layout tailored for executive-level presentation, ensuring critical data is digestible at a glance.

## 🗂️ Project Repository Structure
* `Sales Insigh tReport (3).zip`: Contains the main dataset, project documentation, and the source Power BI (`.pbix`) application file.

## 🛠️ Tech Stack Used
* **Tool:** Microsoft Power BI Desktop
* **Language:** DAX (Data Analysis Expressions) for measures and calculated columns
* **Data Transformation:** Power Query Editor
